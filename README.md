# HPDA — VAST Challenge 2022 (Challenge 2: Patterns of Life)

This repository contains the submission materials and interactive analysis interface for the "Eyes of the City" entry. It includes the written entry form, figures, presentations, and the web application used to explore the results.

**Private repo link:** https://github.com/JohnC21-blip/VAST-Challenge-2-Patterns-of-Life 

## Repository contents (top-level)

| Entry | Description |
| --- | --- |
| `backend/` | Flask app that serves the built frontend from `frontend/dist`. |
| `frontend/` | React + Vite UI with D3/Redux, organized by question (Q1–Q4). |
| `figures/` | Exported figures used in the entry form, grouped by question. |
| `Presentation/` | Slide deck and presentation materials. |
| `Prompts/` | PDF prompt documents for the challenge questions. |
| `index.html` | Completed VAST Challenge entry form (submission document). |
| `instructions.html` | Official submission instructions (for reference). |
